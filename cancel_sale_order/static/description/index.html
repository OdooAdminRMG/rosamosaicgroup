Cancel Confirmed SO / Cancel Processed SO

This application helps to automate process of Cancel Sale Order.

Basic Features
  • This application provides facility to cancel invoice and cancel delivery while cancel sale order, even if invoice was paid and delivery was transfered.
  • "cancel Invoice", "cancel delivery order"
  • User wise configuration.
